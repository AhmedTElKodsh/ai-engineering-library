# Rubric

| Category | Excellent | Passing | Needs Work |
| --- | --- | --- | --- |
| Golden Set | Covers supported, abstained, malformed, and safety cases | Covers core cases | Only happy paths |
| Evaluator | Deterministic, inspectable, category-specific | Computes pass/fail | Uses vague manual judgment |
| Reporting | Names failure categories and counts | Counts pass/fail | Hides why examples failed |
| Production Transfer | Connects failures to release gates | Mentions regression use | Treats evals as paperwork |
