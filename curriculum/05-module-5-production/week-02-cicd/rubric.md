# Rubric: Week 2

| Criterion | Strong Evidence |
| --- | --- |
| Eval evidence | Loads passing and failing eval runs with counts and failure categories. |
| Versioning | Records prompt, model, index, and dataset versions in the gate report. |
| Gate behavior | Fails low pass rates, remaining eval failures, and missing version notes. |
| Reproducibility | Provides exact repo-root commands for unit tests, evals, and gate tests. |
| Reviewability | Gate report includes run ID, pass rate, status, reasons, versions, failures, and checklist. |
| Reflection | Explains why local CI-style evidence comes before cloud deployment. |
