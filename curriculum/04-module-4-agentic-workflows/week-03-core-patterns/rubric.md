# Rubric: Phase 3

| Criterion | Strong Evidence |
| --- | --- |
| Routing | Classifies supported, unsupported, and unsafe requests before tool use. |
| Prompt chaining | Breaks the workflow into named, inspectable steps. |
| Tool boundary | Calls only a deterministic evidence tool and records success or failure. |
| Gate behavior | Answers with evidence, abstains when evidence is missing, and refuses unsafe advice. |
| Traceability | Trace summary exposes route, steps, tools, gate decision, and final response. |
| Reflection | Explains why an explicit workflow is simpler than an autonomous agent for this task. |
