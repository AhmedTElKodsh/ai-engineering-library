# Week 02 Rubric

You are done when:

- all Week 02 tests pass
- you can explain the difference between returning an error value and raising an exception
- you can explain what a context manager guarantees
- you can describe how small classes become client, tool, and agent boundaries
- you can explain why generators prepare you for streaming model responses
- you can add one edge-case test without weakening the existing suite
