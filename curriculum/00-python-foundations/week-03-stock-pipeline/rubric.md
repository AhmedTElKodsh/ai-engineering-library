- `python -m pytest week-03-stock-pipeline/tests -v` passes
- `workbench.py` loads CSV rows, validates data, groups prices, calculates metrics, and renders a safe report

- you can explain how validation, OOP, generators, file I/O, and tests work together

FinAgent readiness check: name how this pipeline extends the Module 1 whole-game workflow, which functions could later become tool calls, which data needs source metadata, and which summary text protects the user from overtrusting the output.
