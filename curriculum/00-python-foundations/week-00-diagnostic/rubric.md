# Week 00 Rubric

You are ready to move on when you can:

- run the setup and diagnostic tests with `python -m pytest`
- identify whether a failure is an import error, syntax error, or assertion failure
- name two Python topics that need practice
- decide whether to take Week 01 slowly or move through it quickly

This unit is a placement gate, not a pass/fail grade.
