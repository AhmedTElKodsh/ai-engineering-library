# Rubric

| Category | Excellent | Passing | Needs Work |
| --- | --- | --- | --- |
| Inspection | Captures status, type, title, selectors, links, and risks | Captures core page facts | Skips important source facts |
| Responsible Collection | Uses robots, terms, attribution, and rate-limit checks | Covers basic permission checks | Treats HTTP 200 as permission |
| Provenance | Defines citation-ready source fields | Keeps source URL and title | Loses source identity |
| Transfer | Explains how the note feeds extraction and RAG | Names the next lab connection | Cannot connect inspection to downstream use |
