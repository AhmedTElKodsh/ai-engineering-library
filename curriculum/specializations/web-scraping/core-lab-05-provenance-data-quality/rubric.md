# Rubric: Core Lab 5

| Criterion | Strong Evidence |
| --- | --- |
| Provenance | Every usable record preserves source URL, source heading, collection timestamp, and extraction assumption. |
| Normalization | Text fields are trimmed and content hashes are deterministic. |

| Reviewability | The provenance table is compact enough for a human to inspect quickly. |
| RAG readiness | The report blocks downstream packaging when quality issues remain. |
| Reflection | Explains why a technically successful scrape may still be unsafe for retrieval. |
