# Rubric

| Category | Excellent | Passing | Needs Work |
| --- | --- | --- | --- |
| Scope | Clear educational use, source limits, non-goals, and safety boundaries | Basic capstone scope | Vague or oversized |
| Eval Harness | Tests citation, abstention, malformed input, and advice refusal | Tests core cases | No repeatable eval |
| Evidence Ledger | Maps artifacts to portfolio claims and reviewer checks | Lists main artifacts | Hides evidence |
| Explanation | Defends tradeoffs and limitations plainly | Explains main choices | Oversells the system |
