# Rubric: FinAgent Risk Signal Extension

| Criterion | Strong | Developing | Needs Work |
| --- | --- | --- | --- |
| Correctness | Risk thresholds and formatting match tests | Most behavior works with one boundary issue | Core behavior missing or hardcoded |
| Test Use | Runs tests after each small change and uses failures as feedback | Runs full suite occasionally | Edits without reading test failures |
| Safety | Summary clearly avoids financial advice | Disclaimer present but weak | Summary sounds like advice |
| Maintainability | Separate helper functions with clear names | Some duplication | Logic tangled in one large function |
| Reflection | Explains risk labels and limitations clearly | Explains only happy path | Cannot explain why the labels exist |
