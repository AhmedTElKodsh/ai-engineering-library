# Rubric

| Category | Excellent | Passing | Needs Work |
| --- | --- | --- | --- |
| Correctness | Meets all expected behavior and edge cases | Meets core behavior | Fails core behavior |
| Reliability | Handles failures clearly | Handles common failures | Crashes or hides errors |
| Explainability | Can explain design and tradeoffs | Can explain main flow | Cannot explain implementation |
| Source Grounding | Uses and cites sources accurately | Uses sources with minor gaps | Makes unsupported claims |
| Security and Privacy | Avoids unsafe inputs and secret leakage | Handles obvious risks | Exposes secrets or unsafe behavior |
| Maintainability | Clear names, small functions, readable tests | Mostly readable | Hard to change safely |
| Test Coverage | Meaningful normal and edge tests | Core tests pass | Missing or ignored tests |
