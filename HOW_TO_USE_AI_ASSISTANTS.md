# How To Use AI Assistants

AI assistants are allowed in this curriculum, but they should make your thinking
clearer rather than replace it.

## Allowed

- ask for an explanation of a failing test
- ask for hints after you have tried the TODO
- ask for examples of a concept using different data
- ask for a review of your explanation
- ask for edge cases you may have missed

## Not Allowed

- paste the whole workbench and ask for a final solution first
- skip reading the tests
- submit generated code you cannot explain
- hide AI help in your evidence notes

## Good Prompt

```text
I tried implementing normalize_terms. This test still fails. Explain what the
test is checking without giving me the final function.
```

## Evidence Note

When you used an assistant, add one sentence:

```text
AI assistance used: I asked for a hint about stable softmax, then wrote and
tested the implementation myself.
```
