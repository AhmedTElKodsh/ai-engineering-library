# Day 05: Control Flow

> You've learned to store and organize data — now teach your code to make decisions.

**Concepts:** if/elif/else, for loops, while loops, break/continue, nested loops | **Difficulty:** ★★☆☆☆ | **Time:** ~30 min
**Builds On:** Day 01 — variables and types, Day 03 — lists and tuples
**Prepares For:** Day 06 — functions and scope, Day 07 — weekly project

## Learning Objectives

By the end of this lesson, you will be able to:
- Implement branching logic using if, elif, and else
- Use for loops to iterate over sequences (lists, ranges)
- Apply while loops for condition-based iteration
- Control loop execution with break and continue
- Process multi-dimensional data using nested loops

## Your Task

Build five functions that demonstrate mastery of control flow: implement the classic FizzBuzz sequence, count steps in the Collatz conjecture, check for palindromes, find prime numbers up to a limit, and sum elements in a 2D matrix.

## How to Work

This daily practice uses a two-step workflow to balance exploration with professional standards:

1. **Draft & Explore** in `exercises.ipynb`: Use the Jupyter Notebook as your interactive scratchpad. It's designed for visual feedback and rapid iteration.
2. **Formalize** in `solution_template.py`: Once your logic is solid, copy your functions into the Python script. This file is your "production" code.
3. **Verify** with the test suite:
   ```bash
   pytest test_solution.py -v
   ```
   **Goal:** Turn all failing tests green. Success reveals the expert model answers!
